PixShift v1.0.0
画像フォーマット変換ツール
==================================================

【動作環境】
  - Windows 10 (バージョン 1809 / ビルド 17763) 以降、または Windows 11
  - x64 (64ビット) 環境

【初回セットアップ（初めて使う PC のみ）】

  Windows App SDK ランタイムのインストールが必要です。
  以下のページから「Windows App SDK 2.1」のインストーラーをダウンロードしてください。

  https://learn.microsoft.com/ja-jp/windows/apps/windows-app-sdk/downloads

  ページ内の「バージョン 2.1」欄にある
  「インストーラー (x64)」をダウンロードして実行してください。
  （ファイル名の例: WindowsAppRuntimeInstall-x64.exe）

  ※ インストール済みの場合やWindows 11 の最新バージョンでは不要なことがあります。
  ※ エラーが出た場合は上記ランタイムをインストールしてから再度起動してください。

【起動方法】
  PixShift.exe をダブルクリックしてください。
  インストール不要でそのまま起動できます。

【対応フォーマット】
  入力: PNG / JPEG / WebP / BMP / TIFF / GIF / HEIC・HEIF
  出力: PNG / JPEG / WebP / BMP / TIFF / GIF / ICO

【主な機能】
  - ドラッグ＆ドロップでファイル・フォルダをまとめて追加
  - バッチ変換（複数ファイルを一括処理）
  - リサイズ（ピクセル指定 / % 指定）
  - ICO 変換時は 16 / 32 / 48 / 256px を自動生成
  - 透明度をもつ画像を JPEG に変換する場合は白背景に合成
  - 出力ファイル名にサフィックス追加 / 連番付き一括リネーム
  - 出力先は入力ファイルと同じ場所の「output」フォルダ（変更可能）
  - 同名ファイルが存在する場合は自動リネーム（上書きなし）

【注意事項】
  - HEIC / HEIF の読み込みには Windows のコーデックが必要です。
    「HEIFイメージ拡張機能」が Microsoft Store から無料で入手できます。
  - 元ファイルは変更・削除されません。
